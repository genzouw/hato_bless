setDateTimeOffset メソッド (SQLServerPreparedStatement)

このメソッドは Microsoft SQL Server JDBC Driver 3.0 で追加されました。

指定した列の値を DateTimeOffset クラス 値に設定します。

構文

public final void setDateTimeOffset(int n, microsoft.sql.DateTimeOffset x)

パラメーター

n

列の 0 から始まる序数です。

x

DateTimeOffset クラス オブジェクトです。

例外

SQLServerException
