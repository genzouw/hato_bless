getResponseBuffering メソッド (ISQLServerStatement)

SQLServerStatement オブジェクトの応答バッファリング モードを取得します。

構文

public final java.lang.String getResponseBuffering()

戻り値

小文字の full または adaptive を含む String です。

説明

adaptive は、必要に応じて最小限のデータをバッファリングすることを示します。

full は、実行時にサーバーから結果全体を読み取ることを示します。

JDBC Driver Version 2.0 および 3.0 では adaptive が既定値です。JDBC Driver Version 2.0 よりも前のバージョンの既定値は full でした。

応答バッファリング モードの使用方法の詳細については、「アダプティブ バッファリングの使用」を参照してください。
