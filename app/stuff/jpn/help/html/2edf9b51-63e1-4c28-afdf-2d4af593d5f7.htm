updateBlob (int, java.io.InputStream, long) メソッド

指定された入力ストリームを使用して、指定された列を更新します。入力ストリームは、指定されたバイト数を持ちます。

構文

public void updateBlob(int columnIndex,
                       java.io.InputStream inputStream,
                                              long length)

パラメーター

columnIndex

列インデックスを示す int です。

inputStream

InputStream オブジェクトです。

length

ストリームの長さを示す long です。

例外

SQLServerException

説明

updateBlob メソッドは、java.sql.ResultSet インターフェイスの updateBlob メソッドで規定されています。
