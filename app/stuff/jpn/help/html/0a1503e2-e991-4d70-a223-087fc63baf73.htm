getObjectInstance メソッド (SQLServerDataSourceObjectFactory)

指定したデータ ソース オブジェクトのインスタンスを取得します。

構文

public java.lang.Object getObjectInstance(java.lang.Object ref,
                                          javax.naming.Name name,
                                          javax.naming.Context c,
                                          java.util.Hashtable h)

パラメーター

ref

Object 値です。

name

オブジェクトの名前です。

c

指定した名前に関連するコンテキストです。

h

オブジェクトの作成に使用する環境です。

戻り値

Object 値です。

例外

java.sql.SQLException

説明

getObjectInstance メソッドは、javax.naming.spi.ObjectFactory インターフェイスの getObjectInstance メソッドで規定されています。
