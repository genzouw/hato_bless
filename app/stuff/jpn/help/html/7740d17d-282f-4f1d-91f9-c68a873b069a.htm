updateRef (java.lang.String, java.sql.Ref) メソッド

渡された列名を使用して、指定された列を java.sql.Ref 値で更新します。

構文

public void updateRef(java.lang.String columnName,
                      java.sql.Ref x)

パラメーター

columnName

列名を含む String です。

x

Ref オブジェクトです。

例外

SQLServerException

説明

updateRef メソッドは、java.sql.ResultSet インターフェイスの updateRef メソッドで規定されています。
