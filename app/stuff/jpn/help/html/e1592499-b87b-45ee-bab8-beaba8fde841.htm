JDBC ドライバーによるパフォーマンスと信頼性の強化

アプリケーション開発において、パフォーマンスと信頼性の強化は常に求められるものです。Microsoft JDBC Driver for SQL Server は、このような要求を実現させる技術を豊富に備えています。

このセクションのトピックでは、JDBC ドライバーを使用する際の、アプリケーションのパフォーマンスと信頼性を強化するさまざまな方法について説明します。
