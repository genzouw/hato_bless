setCatalog メソッド (SQLServerConnection)

渡されたカタログ名を設定し、SQLServerConnection オブジェクトのデータベースの作業用サブスペースを選択します。

構文

public void setCatalog(java.lang.String catalog)

パラメーター

catalog

カタログ名を含む String です。

例外

SQLServerException

説明

setCatalog メソッドは、java.sql.Connection インターフェイスの setCatalog メソッドで規定されています。

catalog 引数は、Microsoft JDBC Driver for SQL Server によって自動的にエスケープされます。このメソッドを使用すると、Connection オブジェクトのカタログ プロパティが設定されます。このプロパティは、他の方法で暗黙的に設定されることはありません。
