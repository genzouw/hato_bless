setClientInfo (java.util.Properties) メソッド

接続のクライアント情報のプロパティの値を設定します。

構文

public void setClientInfo (java.util.Properties properties)

パラメーター

properties

設定するクライアント情報のプロパティの一覧を含む Properties です。

例外

SQLServerException

説明

setClientInfo メソッドは、java.sql.Connection インターフェイスの setClientInfo メソッドで規定されています。

Microsoft JDBC Driver for SQL Server ではクライアント情報のプロパティをサポートしていません。properties 入力パラメーターが空のプロパティ セットを参照しない場合、このメソッドは警告を生成します。つまり、このメソッドはアプリケーションが設定しようとするプロパティへの警告を生成します。アプリケーションは、SQLServerConnection クラスの getWarnings メソッドを使用して各警告を取得する必要があります。
