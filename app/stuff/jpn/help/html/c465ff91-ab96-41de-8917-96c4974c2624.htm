getInt (int) メソッド (ISQLServerResultSet)

SQLServerResultSet オブジェクトの現在の行にある指定された列インデックスの値を、Java プログラミング言語の int として取得します。

構文

public int getInt(int columnIndex)

パラメーター

columnIndex

列インデックスを示す int です。

戻り値

int 値です。

例外

SQLServerException

説明

getInt メソッドは、java.sql.ResultSet インターフェイスの getInt メソッドで規定されています。

このメソッドは、int、smallint、tinyint、bit などの整数値を安全に返すことができる SQL Server データ型でのみサポートされます。このメソッドを他のデータ型で使用すると、例外がスローされます。
