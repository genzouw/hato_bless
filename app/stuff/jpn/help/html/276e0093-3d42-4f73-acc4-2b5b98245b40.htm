getXAConnection (java.lang.String, java.lang.String) メソッド

渡されたユーザー名とパスワードを使用して、物理データベース接続の確立を試みます。

構文

public javax.sql.XAConnection getXAConnection(java.lang.String user,
                                              java.lang.String password)

パラメーター

user

ユーザー名を含む String です。

password

パスワードを含む String です。

戻り値

XAConnection オブジェクトです。

例外

java.sql.SQLException

説明

getXAConnection メソッドは、javax.sql.XADataSource インターフェイスの getXAConnection メソッドで規定されています。

このメソッドは、通常 XA 接続プール実装によって呼び出され、標準の JDBC アプリケーション コードからは呼び出されません。
