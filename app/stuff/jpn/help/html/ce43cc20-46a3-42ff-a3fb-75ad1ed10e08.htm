接続およびデータの取得

Microsoft JDBC Driver for SQL Server を使用する場合は、SQL Server データベースへの接続を確立する 2 つの主な方法があります。1 つは、connection URL の接続プロパティを設定してから、DriverManager クラスの getConnection メソッドを呼び出し、SQLServerConnection オブジェクトを返す方法です。

JDBC ドライバーでサポートされる接続プロパティの一覧については、「接続プロパティの設定」を参照してください。

2 つ目の方法では、SQLServerDataSource クラスの setter メソッドを使用して接続プロパティを設定してから、getConnection メソッドを呼び出し、SQLServerConnection オブジェクトを返します。

このセクションのトピックでは、SQL Server データベースに接続するさまざまな方法について説明し、データを取得するさまざまな手法も示します。
