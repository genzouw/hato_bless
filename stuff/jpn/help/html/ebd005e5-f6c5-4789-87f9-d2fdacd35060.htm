position (java.sql.Blob, long) メソッド

渡されたパターンと開始インデックスに基づいて、BLOB 内の指定されたパターンの位置を返します。

構文

public long position(java.sql.Blob pattern,
                     long start)

パラメーター

pattern

検索するパターンです。

start

検索の開始インデックスです。

戻り値

パターンが見つかった場合は位置の long 値で、見つからなかった場合は -1 です。

例外

SQLServerException

説明

position メソッドは、java.sql.Blob インターフェイスの position メソッドで規定されています。
