getSuperTables メソッド (SQLServerDatabaseMetaData)

データベース内の特定のスキーマで定義されたテーブルの階層の記述を取得します。

このメソッドは、現在 Microsoft JDBC Driver for SQL Server ではサポートされていません。このメソッドを使用すると、常に空の結果セットが返されます。

構文

public java.sql.ResultSet getSuperTables(java.lang.String catalog,
                                         java.lang.String schemaPattern,
                                         java.lang.String tableNamePattern)

パラメーター

catalog

カタログ名を含む String です。

schemaPattern

スキーマ名のパターンを含む String です。

tableNamePattern

テーブル名のパターンを含む String です。

戻り値

SQLServerResultSet オブジェクトです。

例外

SQLServerException

説明

getSuperTables メソッドは、java.sql.DatabaseMetaData インターフェイスの getSuperTables メソッドで規定されています。
