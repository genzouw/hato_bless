getPortNumber メソッド (SQLServerDataSource)

SQL Server との通信に使用される現在のポート番号を返します。

構文

public int getPortNumber()

戻り値

現在のポート番号を表す int 値です。

説明

このポート番号は、SQL Server へのソケット接続を開くときに使用される TCP/IP ポート番号です。portNumber プロパティが設定されていない場合、getPortNumber メソッドは既定値の 1433 を返します。

setPortNumber メソッドは、渡されたポート値に対して値の範囲チェックを行いません。エラーを発生させずに 99999 などの無効なポート番号を渡すことができます。
