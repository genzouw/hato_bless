SQLServerDataSource のメンバー

次の表は、SQLServerDataSource クラスによって公開されるメンバーを示しています。

コンストラクター

名前

説明

SQLServerDataSource ()

SQLServerDataSource クラスの新しいインスタンスを初期化します。

フィールド

なし。

継承されたフィールド

なし。

メソッド

名前

説明

getApplicationIntent

applicationIntent 接続プロパティの値を返します。

getApplicationName

アプリケーション名を返します。

getConnection

SQLServerDataSource オブジェクトが示すデータ ソースとの接続の確立を試みます。

getDatabaseName

データベース名を返します。

getEncrypt

encrypt プロパティが有効であるかどうかを示す Boolean 値を返します。

getDescription

データ ソースの記述を返します。

getFailoverPartner

データベース ミラーリング構成で使用されるフェールオーバー サーバーの名前を返します。

getHostNameInCertificate

SQL Server の SSL (Secure Sockets Layer) 証明書の検証に使用されるホスト名を返します。

getInstanceName

SQL Server インスタンス名を返します。

getLastUpdateCount

lastUpdateCount プロパティが有効であるかどうかを示す boolean 値を返します。

getLockTimeout

データベースがロック タイムアウトを通知するまでに待機する時間 (ミリ秒) を示す int 値を返します。

getLoginTimeout

接続の試行中にこの SQLServerDataSource オブジェクトが待機する秒数を返します。

getLogWriter

すべてのログ メッセージとトレース メッセージで使用される文字出力ストリームを返します。

getMultiSubnetFailover

multiSubnetFailover 接続プロパティの値を返します。

getPacketSize

SQL Server との通信に使用される現在のネットワーク パケット サイズを返します (バイトで指定します)。

getPortNumber

SQL Server との通信に使用される現在のポート番号を返します。

getReference

SQLServerDataSource オブジェクトへの参照を返します。

getResponseBuffering

SQLServerDataSource オブジェクトの応答バッファリング モードを返します。

getSelectMethod

SQLServerDataSource オブジェクトを使用して作成されるすべての結果セットで使用される、既定のカーソルの種類を返します。

getSendStringParametersAsUnicode

サーバーへの文字列パラメーターの UNICODE 形式による送信が有効になっているかどうかを示す Boolean 値を返します。

getSendTimeAsDatetime

SendTimeAsDatetime 接続プロパティの設定を返します。

getServerName

SQL Server を実行しているコンピューターの名前を返します。

getTrustServerCertificate

trustServerCertificate プロパティが有効であるかどうかを示す Boolean 値を返します。

getTrustStore

証明書の trustStore ファイルへのパス (ファイル名を含む) を返します。

getURL

データ ソースへの接続に使用される URL を返します。

getUser

データ ソースへの接続に使用されるユーザー名を返します。

getUseSQLServerBaseDate

useSQLServerBaseDate 接続プロパティの設定を返します。

getWorkstationID

データ ソースへの接続に使用されるクライアント コンピューターの名前を返します。

getXopenStates

XOPEN 互換の状態への SQL 状態の変換が有効になっているかどうかを示す Boolean 値を返します。

isWrapperFor

データ ソース オブジェクトが指定されたインターフェイスのラッパーであるかどうかを示します。

setApplicationIntent

applicationIntent 接続プロパティの値を設定します。

setApplicationName

アプリケーション名を設定します。

setAuthenticationScheme

アプリケーションで使用する統合セキュリティの種類を示します。

setDatabaseName

接続するデータベース名を設定します。

setDescription

データ ソースの記述を設定します。

setEncrypt

encrypt プロパティが有効であるかどうかを示す Boolean 値を設定します。

setFailoverPartner

データベース ミラーリング構成で使用されるフェールオーバー サーバーの名前を設定します。

setHostNameInCertificate

SQL Server の SSL (Secure Sockets Layer) 証明書の検証に使用されるホスト名を設定します。

setInstanceName

SQL Server インスタンス名を設定します。

setIntegratedSecurity

integratedSecurity プロパティが有効であるかどうかを示す Boolean 値を設定します。

setLastUpdateCount

lastUpdateCount プロパティが有効であるかどうかを示す Boolean 値を設定します。

setLockTimeout

データベースがロック タイムアウトを通知するまでに待機する時間 (ミリ秒) を示す int 値を設定します。

setLoginTimeout

接続の試行中に SQLServerDataSource オブジェクトが待機する秒数を設定します。

setLogWriter

すべてのログ メッセージとトレース メッセージで文字出力ストリームが使用されるように設定します。

setMultiSubnetFailover

multiSubnetFailover 接続プロパティの値を設定します。

setPacketSize

SQL Server との通信に使用される現在のネットワーク パケット サイズを設定します (バイトで指定します)。

setPassword

SQL Server への接続に使用されるパスワードを設定します。

setPortNumber

SQL Server との通信に使用されるポート番号を設定します。

setResponseBuffering

SQLServerDataSource オブジェクトを使用して作成された接続の応答バッファリング モードを設定します。

setSelectMethod

SQLServerDataSource オブジェクトを使用して作成されるすべての結果セットで使用される、既定のカーソルの種類を設定します。

setSendStringParametersAsUnicode

サーバーへの文字列パラメーターの UNICODE 形式による送信が有効になっているかどうかを示す Boolean 値を設定します。

setSendTimeAsDatetime

java.sql.Time 値をサーバーに送信する方法を指定します。

setServerName

SQL Server を実行しているコンピューターの名前を設定します。

setTrustServerCertificate

trustServerCertificate プロパティが有効であるかどうかを示す Boolean 値を設定します。

setTrustStore

証明書の trustStore ファイルへのパス (ファイル名を含む) を設定します。

setTrustStorePassword

trustStore データの整合性を確認するために使用するパスワードを設定します。

setURL

データ ソースへの接続に使用される URL を設定します。

setUser

データ ソースへの接続に使用されるユーザー名を設定します。

setWorkstationID

データ ソースへの接続に使用されるクライアント コンピューターの名前を設定します。

setXopenStates

XOPEN 互換の状態への SQL 状態の変換が有効になっているかどうかを示す Boolean 値を設定します。

unwrap

指定されたインターフェイスを実装するオブジェクトを返します。このメソッドから返されたオブジェクトを使用することで、Microsoft JDBC Driver for SQL Server 固有のメソッドにアクセスできます。
