setTimestamp (int, java.sql.Timestamp, java.util.Calendar) メソッド

渡された Calendar オブジェクトを使用して、指定されたパラメーターを、渡されたタイムスタンプの値に設定します。

構文

public final void setTimestamp(int n,
                               java.sql.Timestamp x,
                               java.util.Calendar cal)

パラメーター

n

パラメーターの番号を示す int です。

x

Timestamp オブジェクトです。

cal

Calendar オブジェクトです。

例外

SQLServerException

説明

setTimestamp メソッドは、java.sql.PreparedStatement インターフェイスの setTimestamp メソッドで規定されています。
