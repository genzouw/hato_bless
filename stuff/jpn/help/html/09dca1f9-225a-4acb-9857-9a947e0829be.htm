setTimestamp (java.lang.String, java.sql.Timestamp, java.util.Calendar) メソッド

渡された Calendar オブジェクトを使用して、指定されたパラメーターを、渡されたタイムスタンプの値に設定します。

構文

public void setTimestamp(java.lang.String sCol,
                         java.sql.Timestamp x,
                         java.util.Calendar c)

パラメーター

sCol

パラメーターの名前を含む String です。

x

Timestamp オブジェクトです。

c

Calendar オブジェクトです。

例外

SQLServerException

説明

setTimestamp メソッドは、java.sql.CallableStatement インターフェイスの setTimestamp メソッドで規定されています。
