getColumnDisplaySize メソッド (SQLServerResultSetMetaData)

指定された列の通常の最大幅を表す文字数を返します。

構文

public int getColumnDisplaySize(int column)

パラメーター

column

列インデックスを示す int です。

戻り値

最大幅を示す int です。最大幅が不明の場合は、0 を返します。

例外

SQLServerException

説明

getColumnDisplaySize メソッドは、java.sql.ResultSetMetaData インターフェイスの getColumnDisplaySize メソッドで規定されています。

Microsoft SQL Server JDBC Driver 3.0 では、COLUMN_SIZE 列の動作が変更されています。詳細については、SQLServerDatabaseMetaData.getColumns を参照してください。
