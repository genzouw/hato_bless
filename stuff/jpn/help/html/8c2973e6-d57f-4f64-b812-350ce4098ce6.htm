getBytes (int) メソッド

パラメーターに渡されたインデックスを使用して、指定されたパラメーターの値を byte 配列として取得します。

構文

public byte[] getBytes(int index)

パラメーター

index

パラメーターのインデックスを示す int です。

戻り値

byte 値の配列です。

例外

SQLServerException

説明

以前のバージョンの Microsoft JDBC Driver for SQL Server では、SQLServerCallableStatement.getBytes を使用してバイト配列と SQL Server データ型の date、time、datetime2、datetimeoffset との間の変換を実行できました。新しいバージョンでは、これらのデータ型に対してこのメソッドを使用すると、変換がサポートされていないことを示す例外が発生します。

getBytes メソッドは、java.sql.CallableStatement インターフェイスの getBytes メソッドで規定されています。
