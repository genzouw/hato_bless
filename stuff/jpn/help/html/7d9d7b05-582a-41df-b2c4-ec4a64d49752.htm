getDateTimeOffset メソッド (SQLServerCallableStatement)

このメソッドは Microsoft SQL Server JDBC Driver 3.0 で追加されました。

パラメーターに渡されたインデックスを使用して、指定されたパラメーターの値を Java プログラミング言語の DateTimeOffset クラス オブジェクトとして取得します。

DateTimeOffset クラス パラメーターは SQLServerCallableStatement.setDateTimeOffset を使用して設定できます。
