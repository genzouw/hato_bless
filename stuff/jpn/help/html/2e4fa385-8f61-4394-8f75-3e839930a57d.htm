createStatement (int, int, int) メソッド

渡された結果セットの種類、同時実行、および保持機能の SQLServerResultSet オブジェクトを生成する SQLServerStatement オブジェクトを作成します。

構文

public java.sql.Statement createStatement(int nType,
                                          int nConcur,
                                          int nHold)

パラメーター

resultSetType

結果セットの種類を表す int 値です。

nConcur

結果セットの同時実行の種類を表す int 値です。

nHold

保持機能を表す int 値です。

戻り値

Statement オブジェクトです。

例外

SQLServerException

説明

createStatement メソッドは、java.sql.Connection インターフェイスの createStatement メソッドで規定されています。
