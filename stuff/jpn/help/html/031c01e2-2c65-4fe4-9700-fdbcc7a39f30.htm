ISQLServerConnection インターフェイス

Microsoft SQL Server データベースへの JDBC 接続を表します。このインターフェイスは SQL Server JDBC Driver 3.0 で追加されました。

パッケージ : com.microsoft.sqlserver.jdbc

継承: java.sql.Connection

説明

このインターフェイスは SQLServerConnection クラス によって実装されています。
