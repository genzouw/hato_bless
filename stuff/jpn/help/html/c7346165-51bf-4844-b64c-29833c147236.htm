commit メソッド (ISQLServerConnection)

前回のコミットまたはロールバック以降のすべての変更を永続的にして、SQLServerConnection オブジェクトによって現在保持されているデータベース ロックをすべて解除します。

構文

public void commit()

例外

SQLServerException

説明

commit メソッドは、java.sql.Connection インターフェイスの commit メソッドで規定されています。

このメソッドは、自動コミット モードが無効になっている場合にのみ使用してください。

クライアントが手動トランザクションを開始した後、SQL Server が何らかの理由でその手動トランザクションをロールバックした場合、このメソッドは失敗して例外をスローすることに注意してください。たとえば、クライアントが明示的に ROLLBACK TRANSACTION を呼び出すストアド プロシージャを呼び出した後で commit メソッドを呼び出すと、例外がスローされます。また、クライアントが開始した手動トランザクションのロールバック条件を満たす重大度 16 以上のエラーが SQL Server で発生した場合、それ以降の commit メソッドの呼び出しでは例外がスローされます。
