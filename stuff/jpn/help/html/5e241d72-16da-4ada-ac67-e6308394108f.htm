ドライバー バージョンの取得

インストールされている Microsoft JDBC Driver for SQL Server のバージョンは、次の方法で確認できます。

  • SQLServerDatabaseMetaData の getDriverMajorVersion メソッド、getDriverMinorVersion メソッド、または getDriverVersion メソッドを呼び出す

  • 製品付属の readme.txt ファイルに記載されているバージョンを確認する

また、SQLServerDatabaseMetaData クラスの getDriverName メソッドを呼び出すと、JDBC ドライバー名が返されます。返されるドライバー名は "Microsoft JDBC Driver 4.0 for SQL Server" です。

SQLServerDatabaseMetaData クラスの各メソッドを呼び出した場合の出力例を次に示します。

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

"xxx.x" は最終バージョン番号です。
