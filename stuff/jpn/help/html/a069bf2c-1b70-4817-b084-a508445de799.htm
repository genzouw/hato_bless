SSTRANSTIGHTLYCPLD

XA ブランチ トランザクション ID (XID) は異なるが、グローバル トランザクション ID (GTRID) が同じである、密に結合された XA トランザクションを許可するために使用します。

メソッド

名前

説明

commit

渡された Xid オブジェクトによって指定されたグローバル トランザクションをコミットします。

end

トランザクション ブランチのために実行していた処理を終了します。

forget

リソース マネージャーに対し、ヒューリスティックに決着されたトランザクション ブランチを無視するよう指示します。

getTransactionTimeout

SQLServerXAResource オブジェクトに設定された現在のトランザクション タイムアウト値を取得します。

isSameRM

対象オブジェクトが表すリソース マネージャー インスタンスと、渡された XAResource オブジェクトが表すリソース マネージャー インスタンスとが同一であるかどうかを示します。

prepare

渡された Xid オブジェクトによって指定されたトランザクションのコミットに対する準備を行うように、リソース マネージャーに要求します。

recover

準備されたトランザクション ブランチの一覧を、リソース マネージャーから取得します。

rollback

トランザクション ブランチのために実行した処理をロールバックするように、リソース マネージャーに要求します。

setTransactionTimeout

SQLServerXAResource オブジェクトの現在のトランザクション タイムアウト値を設定します。

start

Xid オブジェクトで指定されたトランザクション ブランチのために処理を開始します。
