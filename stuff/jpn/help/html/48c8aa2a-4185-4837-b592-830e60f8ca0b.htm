setNClob (int, java.sql.NClob) メソッド

指定されたパラメーターを、指定された NClob オブジェクトに設定します。

構文

public final void setNClob(int parameterIndex,
              java.sql.NClob value)

パラメーター

parameterIndex

パラメーターのインデックスを示す int です。

value

パラメーター値を示す NClob オブジェクトです。

例外

SQLServerException

説明

setNClob メソッドは、java.sql.PreparedStatement インターフェイスの setNClob メソッドで規定されています。
