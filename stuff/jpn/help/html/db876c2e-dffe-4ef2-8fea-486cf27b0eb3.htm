execute メソッド (ISQLServerPreparedStatement)

SQLServerPreparedStatement オブジェクトの SQL ステートメントを実行します。すべての種類の SQL ステートメントを実行することができます。

オーバーロードの一覧

名前

説明

execute ()

SQLServerPreparedStatement オブジェクトの SQL ステートメントを実行します。すべての種類の SQL ステートメントを実行することができます。

execute (java.lang.String)

渡された SQL ステートメントを実行します。このステートメントは、複数の結果を返す場合があります。
