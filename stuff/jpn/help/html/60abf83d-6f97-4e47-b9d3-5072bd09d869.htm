getDateTimeOffset(int) (SQLServerResultSet)

このメソッドは Microsoft SQL Server JDBC Driver 3.0 で追加されました。

パラメーターに渡されたインデックスを使用して、指定された列の値を Java プログラミング言語の DateTimeOffset クラス オブジェクトとして取得します。

構文

public microsoft.sql.DateTimeOffset getDateTimeOffset(int columnIndex)

パラメーター

columnIndex

列の序数。

戻り値

DateTimeOffset クラス オブジェクトです。

例外

SQLServerException

説明

SQLServerResultSet.updateDateTimeOffset を使用して、DateTimeOffset クラス 値を更新できます。
