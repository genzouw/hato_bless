getColumnClassName メソッド (SQLServerResultSetMetaData)

SQLServerResultSet クラスの getObject メソッドを呼び出して列から値を取得する場合、インスタンスが生成される Java クラスの完全修飾名を返します。

構文

public java.lang.String getColumnClassName(int column)

パラメーター

column

列インデックスを示す int です。

戻り値

クラスの完全修飾名を含む String です。

例外

SQLServerException

説明

getColumnClassName メソッドは、java.sql.ResultSetMetaData インターフェイスの getColumnClassName メソッドで規定されています。
