ISQLServerCallableStatement インターフェイス

JDBC の呼び出し可能ステートメントを表します。このインターフェイスは SQL Server JDBC Driver 3.0 で追加されました。

パッケージ: com.microsoft.sqlserver.jdbc

継承: java.sql.CallableStatement、ISQLServerPreparedStatement

構文

public interface ISQLServerCallableStatement

説明

このインターフェイスは SQLServerCallableStatement クラス によって実装されています。

このインターフェイスでは、次の Microsoft JDBC Driver for SQL Server 固有のメソッドが公開されます。
