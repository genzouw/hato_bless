prepareStatement メソッド (ISQLServerConnection)

パラメーター化された SQL ステートメントをデータベースに送信するための SQLServerPreparedStatement オブジェクトを作成します。

オーバーロードの一覧

名前

説明

prepareStatement (java.lang.String)

パラメーター化された SQL ステートメントをデータベースに送信するための SQLServerPreparedStatement オブジェクトを作成します。

prepareStatement (java.lang.String, int)

パラメーター化された SQL ステートメントをデータベースに送信するための SQLServerPreparedStatement オブジェクトを作成します。自動生成キーを取得する機能を備えています。

prepareStatement (java.lang.String, int[])

パラメーター化された SQL ステートメントをデータベースに送信するための SQLServerPreparedStatement オブジェクトを作成します。渡される配列で指定される自動生成キーを返すことができます。

prepareStatement (java.lang.String, int, int)

渡された結果セットの種類および同時実行の SQLServerResultSet オブジェクトを生成する SQLServerPreparedStatement オブジェクトを作成します。

prepareStatement (java.lang.String, int, int, int)

渡された結果セットの種類、同時実行、および保持機能の SQLServerResultSet オブジェクトを生成する SQLServerPreparedStatement オブジェクトを作成します。

prepareStatement (java.lang.String, java.lang.String[])

パラメーター化された SQL ステートメントをデータベースに送信するための SQLServerPreparedStatement オブジェクトを作成します。
