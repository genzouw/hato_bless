createStatement メソッド (ISQLServerConnection)

SQL ステートメントをデータベースに送信するための SQLServerStatement オブジェクトを作成します。

オーバーロードの一覧

名前

説明

createStatement ()

SQL ステートメントをデータベースに送信するための SQLServerStatement オブジェクトを作成します。

createStatement (int, int)

渡された結果セットの種類および同時実行の SQLServerResultSet オブジェクトを生成する SQLServerStatement オブジェクトを作成します。

createStatement (int, int, int)

渡された結果セットの種類、同時実行、および保持機能の SQLServerResultSet オブジェクトを生成する SQLServerStatement オブジェクトを作成します。
