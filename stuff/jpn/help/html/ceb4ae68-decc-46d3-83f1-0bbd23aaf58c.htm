getFetchDirection メソッド (SQLServerStatement)

行をフェッチする方向をデータベース テーブルから取得します。この方向は、SQLServerStatement オブジェクトから生成される結果セットの既定の方向となります。

このメソッドは、現在 Microsoft JDBC Driver for SQL Server では実装されていません。そのため、このメソッドは常に FETCH_UNKNOWN を返します。

構文

public final int getFetchDirection()

戻り値

setFetchDirection メソッドによって指定されるフェッチ方向を示す int です。

例外

SQLServerException

説明

getFetchDirection メソッドは、java.sql.Statement インターフェイスの getFetchDirection メソッドで規定されています。
