getLong (int) メソッド

パラメーターに渡されたインデックスを使用して、指定されたパラメーターの値を Java プログラミング言語の long として取得します。

構文

public long getLong(int index)

パラメーター

index

パラメーターのインデックスを示す int です。

戻り値

long 値です。

例外

SQLServerException

説明

getLong メソッドは、java.sql.CallableStatement インターフェイスの getLong メソッドで規定されています。

このメソッドは、bigint、int、smallint、tinyint、bit などの整数値を安全に返すことができる SQL Server データ型でのみサポートされています。このメソッドを他のデータ型で使用すると、例外がスローされます。
