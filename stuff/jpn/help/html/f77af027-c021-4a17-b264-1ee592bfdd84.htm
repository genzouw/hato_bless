isWrapperFor メソッド (SQLServerDataSource)

データ ソース オブジェクトが指定されたインターフェイスのラッパーであるかどうかを示します。

構文

public boolean isWrapperFor(Class iface)

パラメーター

iface

インターフェイスを定義する class です。

戻り値

このオブジェクトがインターフェイスを実装しているか、インターフェイスを実装しているオブジェクトをラップしている場合は true です。それ以外の場合は false です。

例外

SQLServerException

説明

isWrapperFor メソッドと unwrap メソッドは、JDBC 4.0 仕様で導入された java.sql.Wrapper インターフェイスで定義されています。

このメソッドが true を返す場合、同じ引数を使用した unwrap の呼び出しは成功します。

詳細については、「ラッパーとインターフェイス」を参照してください。
