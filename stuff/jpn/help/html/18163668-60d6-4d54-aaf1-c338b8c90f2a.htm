getDatabaseMinorVersion メソッド (SQLServerDatabaseMetaData)

基本データベースのマイナー バージョン番号を取得します。

構文

public int getDatabaseMinorVersion()

戻り値

データベースのマイナー バージョンを示す int です。

例外

SQLServerException

説明

getDatabaseMinorVersion メソッドは、java.sql.DatabaseMetaData インターフェイスの getDatabaseMinorVersion メソッドで規定されています。
