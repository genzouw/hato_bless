getURL (int) メソッド

パラメーターに渡されたインデックスを使用して、指定されたパラメーターの値を Java プログラミング言語の URL オブジェクトとして取得します。

構文

public java.net.URL getURL(int n)

パラメーター

n

パラメーターのインデックスを示す int です。

戻り値

URL オブジェクトです。

例外

SQLServerException

説明

getURL メソッドは、java.sql.CallableStatement インターフェイスの getURL メソッドで規定されています。
