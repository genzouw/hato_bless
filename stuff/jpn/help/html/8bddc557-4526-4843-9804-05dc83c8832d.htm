registerOutParameter (java.lang.String, int, int) メソッド

指定された名前の OUT パラメーターを、渡された JDBC 型と小数点以下桁数に登録します。

構文

public void registerOutParameter(java.lang.String s,
                                 int n,
                                 int n1)

パラメーター

s

パラメーターの名前を含む String です。

sqlType

java.sql.Types で定義される JDBC 型のコードです。

scale

小数点以下の桁数を示す int です。

例外

SQLServerException

説明

registerOutParameter メソッドは、java.sql.CallableStatement インターフェイスの registerOutParameter メソッドで規定されています。
