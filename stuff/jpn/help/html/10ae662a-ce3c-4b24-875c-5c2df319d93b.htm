executeUpdate メソッド (SQLServerStatement)

渡された SQL ステートメントを実行します。ステートメントは INSERT、UPDATE、DELETE、または SQL DDL ステートメントのような何も返さない SQL ステートメントが可能です。Microsoft SQL Server JDBC Driver 3.0 以降、executeUpdate は、MERGE 操作で更新された行の正確な数を返します。

オーバーロードの一覧

名前

説明

executeUpdate (java.lang.String)

渡された SQL ステートメントを実行します。ステートメントは INSERT、UPDATE、DELETE、MERGE、または SQL DDL ステートメントのような何も返さない SQL ステートメントが可能です。

executeUpdate (java.lang.String, int)

渡された SQL ステートメントを実行し、SQLServerStatement オブジェクトによって自動生成されたキーを検索可能にするかどうかについて、渡されたフラグを使用して Microsoft JDBC Driver for SQL Server に通知します。

executeUpdate (java.lang.String, int[])

渡された SQL ステートメントを実行し、渡された配列に示される自動生成キーを検索可能にするように、JDBC ドライバーに通知します。

executeUpdate (java.lang.String, java.lang.String[])

渡された SQL ステートメントを実行し、渡された配列に示される自動生成キーを検索可能にするように、JDBC ドライバーに通知します。
